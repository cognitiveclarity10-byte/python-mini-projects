This project is a beginner-friendly physics simulation built using **Python and Pygame**.  
It demonstrates how a ball moves and bounces inside a boxed area using basic physics concepts such as velocity, collision detection, and screen coordinates.

The main goal of this project is to **learn physics programming step by step**, focusing on understanding how movement and collisions actually work instead of relying on game engines or shortcuts.

## ‚ú® Features

- Pygame window with a centered box
- Four walls created using `pygame.Rect`
- Ball movement using horizontal and vertical velocity

- Smooth bouncing without sticking or vibration
- Consistent motion using frame rate control
